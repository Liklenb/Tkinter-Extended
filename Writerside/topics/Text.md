# Text

Display the given text.

## Usage

```python
Text(
    text = 'Here is some text'
)
```

## Properties

### text

```str``` (required)

The text to display.

### color

[```Color```](Color.md) (default: ```Colors.black```)

The color of the text.

### font_size

```int``` (default: ```15```)

The size of the text.

## Constraints

`Text` isn't influenced by its parent constraints.
